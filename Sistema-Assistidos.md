# :star: Sistema de Assistidos

## :bust_in_silhouette: Responsável

Sergio – RA

## :pushpin: Descrição

Permite marcar filmes como assistidos e salvar essa informação no navegador.

## :white_check_mark: Funcionalidades implementadas

* Botão "Marcar como assistido"
* Alteração visual do status
* Persistência com localStorage

## :file_folder: Arquivos modificados

* script.js

## :test_tube: Como testar

Clicar no botão de assistido e recarregar a página para verificar se o estado foi salvo.

## :twisted_rightwards_arrows: Integração

Integrado com a listagem de filmes.
