# 🎬 Catálogo de Filmes e Séries

## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de uma aplicação web de catálogo de filmes e séries.
O sistema permite visualizar, buscar e organizar conteúdos, oferecendo uma experiência simples e interativa para o usuário.

---

## 🎯 Objetivo

Desenvolver uma aplicação colaborativa utilizando GitHub, aplicando conceitos de:

* Versionamento com Git
* Trabalho em equipe com branches
* Organização de código
* Desenvolvimento front-end

---

## ⚙️ Funcionalidades

### ✅ Funcionalidades 

* Listagem de filmes/séries
* Busca por título
* Marcar como:

  * Assistido
  * Quero assistir
* Visualização de detalhes
* Interface responsiva básica

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
